# Design System: riff.

Este documento detalha a identidade visual do app **riff.**, extraída das telas de boas-vindas, login e criação de perfil.

## 🎨 Paleta de Cores

O riff. utiliza um tema **Dark Mode** profundo com cores de destaque vibrantes para ações principais.

### Cores de Superfície
- **Background:** `#131313` (Preto profundo para foco no conteúdo)
- **Surface-Low:** `#1B1B1B` (Para campos de entrada e containers secundários)
- **Surface-Bright:** `#393939` (Bordas e divisores sutis)

### Cores de Destaque (Brand Colors)
- **Laranja Riff:** `#FF5722` (Cor principal para botões primários, ícones e destaques de seleção)
- **Verde Spotify:** `#1DB954` (Usado exclusivamente para a integração com Spotify)

### Tipografia
- **Cor do Texto (High Emphasis):** `#FFFFFF` (Títulos e textos importantes)
- **Cor do Texto (Medium Emphasis):** `#A0A0A0` (Subtítulos, placeholders e labels)

---

## 🔡 Tipografia

A tipografia é moderna e limpa, otimizada para legibilidade em interfaces digitais.

- **Fonte Principal:** `Plus Jakarta Sans`
- **Títulos:** Bold (Negrito) com espaçamento generoso.
- **Corpo de Texto:** Medium/Regular.
- **Botões:** Semibold, em caixa alta (uppercase) para o CTA principal.

---

## 🧩 Elementos Visuais e Componentes

### Botões
- **Primário:** Fundo laranja sólido (`#FF5722`), texto branco, bordas totalmente arredondadas (`rounded-full`).
- **Secundário/Social:** Fundo verde sólido (`#1DB954`) para Spotify; contorno sutis para outras opções.
- **Outline:** Borda fina com fundo transparente para ações secundárias.

### Inputs e Tags
- **Campos de Texto:** Fundo escuro (`#1B1B1B`) com bordas arredondadas. O estado ativo recebe uma borda laranja.
- **Tags de Subgênero:** Formato pílula. Tags selecionadas possuem borda laranja; não selecionadas possuem borda e texto em cinza médio.

### Assets de Imagem
- **Logotipo:** Tipografia customizada "riff." em branco com um ponto laranja ao final.
- **Avatar de Usuário:** Placeholder circular com borda pontilhada e link para upload.

---

*Este guia serve como referência para manter a consistência visual em futuras telas do ecossistema riff.*
